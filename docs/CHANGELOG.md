## [0.0.1-rtw.8] - 2020-09-12

### Added
- this branch contains a simple website, serving as a reception for our portfolio of websites.

Whichever url you visit (hotspotbnb.com, scottishbay.online, etc) you arrive at this webpage.
- this was the only way to achieve a scenarion where the website is accessible via its corresponding domain,
- while all being hosted in a single monorepo, without having to pay netlify fees every time a change is made to the websites
- and without losing ssl continuity, without using mask or url forwarding
