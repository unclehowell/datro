Release Notes and Notices
**************************

DATRO re-writes the documentation for each of its application, presenting them to our users in a uniform format e.g. reStructuredText, PDF and HTML.
The semantic version corresponds to the re-writen documentation and not the origional authors release version. 
This section provides information about what is new or changed, including urgent issues, documentation updates, maintenance and new releases.
